# 把Red编译成Red System代码


## 例子：test.red

```
Red [ ]

; 数值
a: 1
b: 2 + a
print b

; 字符串
s: "hello"
prin s
print " world"

; 函数定义与调用
inc: func [n][n + 1]
b: inc b
print b

```
