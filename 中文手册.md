#入门：
* 编辑一个文本文件名为hello.red内容如下

```
Red []`

print "hello　world"
```
运行：
./red-061  hello.red 
输出
```hello world```

注意：仅限于Red代码，Red/System代码不可以这样运行，只能用-c 编译成可执行文件
